# Website for The Power Up Project

Site for Power Up Project, a 501(c)(3) non-profit based in Illinois.

Setup:

Create .env file in the root directory (/power-up-project-site):

```
SESSION_SECRET=create-a-session-secret (https://randomkeygen.com/)
PORT_NUM=XXXX (ex: 8080)
MONGO_URI=Ask-Jack
```

Open a terminal in the root directory (/power-up-project-site) and run:

```
npm i
```
